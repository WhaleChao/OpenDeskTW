# ONLYOFFICE AI 台灣繁中相容語系

此目錄只隨附 `zh-TW` 語系檔。修復程序會從使用者電腦上已安裝的 ONLYOFFICE Desktop Editors 複製其內建 AI 外掛，改用新的外掛 GUID，再覆寫這兩份台灣繁體中文語系；不會修改 `/Applications` 或 `Program Files` 內已簽章的原程式。

翻譯鍵以 ONLYOFFICE AI plugin 3.2.2 的 `zh-CN` 語系為基礎，經繁體轉換及台灣用語校訂。原 AI plugin 與衍生語系依 GNU AGPL v3 授權；原介面文字另依其上游聲明適用 CC BY-SA 4.0。上游版權屬 Ascensio System SIA。全能文件工作台的修改與新增翻譯依 AGPL-3.0-or-later 發佈。

內建 AI 與 MAGI 是兩套不同功能：繁中相容副本顯示為「其他 AI 模型」，本機 MAGI V2／V3 顯示於獨立「MAGI」頁籤。
