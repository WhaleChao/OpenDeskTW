# 全能文件工作台繁中寫作工具

這是全能文件工作台隨附的本機 ONLYOFFICE 系統外掛。重新開啟 ONLYOFFICE 後，「常用」會集中分散對齊、智慧補齊、中文標題重編與 MAGI 摘要；「全能文件」提供台灣標點整理、數字字級、台灣字型與快捷鍵提示；「MAGI」頁籤則保留摘要、校對、結構與完整檢查。

- 一般繁中與排版工具不讀取、上傳或保存文件內容。
- 只有使用者按下「MAGI」分析按鈕時，才會把選取文字（未選取時為目前文件文字）送到 `127.0.0.1` 的全能文件工作台橋接；不會開啟網頁，也不直接連線外部網站。
- 橋接使用每次啟動隨機產生的權杖，只接受本機 ONLYOFFICE 外掛來源，並自動相容唯一運作的 MAGI V2 或 V3。
- 只在使用者按下按鈕時處理目前選取的文字或句子。
- 智慧補齊會依巢狀順序辨識 `（`、`「`、`【`、`〔`、`《` 等開頭，補上相符的 `）`、`」`、`】`、`〕`、`》`；輸入引號時會依未閉合層級即時選擇 `』` 或 `」`。
- 分散對齊依每個實際排版行計算平均字距，讓文字在 ONLYOFFICE 畫面與輸出中填滿行寬；也可按 Windows／Linux `Ctrl+Shift+J` 或 macOS `⇧⌘J`。
- 「台灣字型」可精確套用新細明體 `PMingLiU` 或細明體 `MingLiU`，不受作業系統字型選單是否列出的限制。
- 格式複製／套用使用 `Ctrl+Alt+C／V` 或 macOS `⌘⌥C／V`，直接接到目前文件的格式快取與單次復原交易。
- Word 相容層補上 1／2／1.5 倍行距、標題 1–3、一般樣式、上下標、凸排、追蹤修訂與新增註解快捷鍵。
- 中文標題重編可在「常用」直接執行，或按 `Ctrl+Alt+Shift+R／⌥⇧⌘R`；只處理段落開頭，內文中的「壹、一、（一）」不會誤判。
- 全能文件工作台會安裝到使用者資料夾，不修改 ONLYOFFICE 已簽章的應用程式本體。
- ONLYOFFICE 原有 AI 不是 MAGI；修復程序另建立顯示為「其他 AI 模型」的台灣繁中相容副本，與「MAGI」頁籤清楚分開。
