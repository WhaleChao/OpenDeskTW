# OpenDesk TW 繁中寫作工具

這是 OpenDesk TW 隨附的本機 ONLYOFFICE 系統外掛。重新開啟 ONLYOFFICE 後，文件編輯器會出現「OpenDesk TW」工具列，提供分散對齊、成對標點智慧補齊、台灣標點整理與快捷鍵提示。

- 不讀取、上傳或保存文件內容。
- 不存取網路或作業系統檔案。
- 只在使用者按下按鈕時處理目前選取的文字或句子。
- 智慧補齊會依巢狀順序辨識 `（`、`「`、`【`、`〔`、`《` 等開頭，補上相符的 `）`、`」`、`】`、`〕`、`》`。
- 分散對齊呼叫目前文件的原生段落對齊 API。
- OpenDesk 會安裝到使用者資料夾，不修改 ONLYOFFICE 已簽章的應用程式本體。
