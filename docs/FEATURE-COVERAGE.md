# 全能文件工作台 2.6.0 功能覆蓋

這份清單是發行閘門，不以「有按鈕」代表完成。狀態分為：

- **已實作並自動驗證**：工作台自己的程式會實際產生、修改、重新開啟並檢查結果。
- **由編輯引擎提供**：在 ONLYOFFICE 或 LibreOffice 內使用；工作台負責繁中、開啟、備份與相容性檢查。
- **相容保留／警示**：能避免破壞原內容，但沒有開源等價執行環境。

## MAGI V2／V3

工作台只連線目前這台電腦上唯一正在運作的 MAGI V2 或 V3，使用相容封包送出摘要、校對、結構或風險分析，結果直接顯示在 App 並可複製。ONLYOFFICE 內另有獨立「MAGI」頁籤，選取文字優先、整份文件備援，透過每次啟動重建權杖的本機橋接取得結果。兩版同時運作或協定無法判斷時會停止送出，不會改開網頁。

ONLYOFFICE 原有的通用 AI 功能保留為「其他 AI 模型」，採獨立 GUID 的台灣繁中相容副本；它不是 MAGI，需要另外設定模型。上游簡體 AI 頁籤只會在繁中副本安裝成功後隱藏，避免功能消失或名稱混淆。

## 無法等價重現或僅能警示

- Microsoft 365 雲端共同編輯、Copilot、IRM／Purview、SharePoint 工作流程。
- COM／VSTO 增益集、ActiveX、完整 VBA 執行環境與依賴 Windows Office 物件模型的自動化。
- 部分複雜 SmartArt、3D 模型、動作路徑及只存在於 Microsoft 排版引擎的逐像素結果。
- PDF 轉可編輯 Office 的複雜多欄版面只能盡量重排；PPTX 採頁面影像保版並把擷取文字放入備忘稿。
- PDF/A、PDF/X、PDF/UA 的完整修復與標準認證需要 ICC、結構樹修復及外部合規驗證器；目前提供預檢與明確警示，不宣稱通過認證。
- 數位簽章的「完整」可在本機驗證；「信任」仍取決於憑證鏈與作業系統／組織信任庫。
